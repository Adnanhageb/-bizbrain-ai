# BizBrain AI - تطبيق تحليل البيانات الذكي

## نظرة عامة
BizBrain AI هو تطبيق ويب تفاعلي لتحليل بيانات الشركات باستخدام الذكاء الاصطناعي. يتيح للمستخدمين رفع ملفات Excel/CSV من أنظمة ERP وتحليلها تلقائياً.

## الميزات
- 📊 **تحليل ذكي تلقائي** - يكتشف الأعمدة ويحسب المؤشرات الرئيسية
- 💬 **محادثة ذكية** - اسأل بالعربية واحصل على إجابات فورية
- 📈 **رسوم بيانية تفاعلية** - اتجاهات الإيرادات، أفضل المنتجات والعملاء
- 🚨 **تنبيهات ذكية** - كشف المخاطر والانحرافات تلقائياً
- 📄 **تقارير PDF** - تصدير تقارير احترافية

## طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
cd bizbrain-ai
npm install
```

### 2. تشغيل التطبيق
```bash
npm run dev
```

### 3. فتح المتصفح
افتح: http://localhost:3000

## تنسيق البيانات المدعوم
يدعم النظام ملفات Excel تحتوي على أعمدة مثل:
- الإيرادات / Revenue
- التكاليف / Cost
- التاريخ / Date
- المنتج / Product
- العميل / Customer
- الفرع / Branch
- الكمية / Quantity
- الربح / Profit

## التقنيات المستخدمة
- React 18 + Vite
- Tailwind CSS
- Recharts (رسوم بيانية)
- SheetJS/xlsx (قراءة Excel)
- Lucide React (أيقونات)
